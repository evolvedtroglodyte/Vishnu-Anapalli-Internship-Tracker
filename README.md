# Summer 2026 Internship Tracker & Resume Matcher

A comprehensive internship tracking tool with integrated AI-powered resume matching.

**Live Site:** [https://evolvedtroglodyte.github.io/Vishnu-Anapalli-Internship-Tracker/](https://evolvedtroglodyte.github.io/Vishnu-Anapalli-Internship-Tracker/)

## Features

### Internship Tracker
- **380 verified Summer 2026 internship roles** across 5 categories:
  - **Texas** (34 roles) - Local opportunities in TX
  - **Business Tech / Analytics** (62 roles) - Data Analyst, Business Analyst, Data Science, BI
  - **Software Engineering / Tech** (250 roles) - SWE/CS positions  
  - **Quant / Trading** (8 roles) - Elite trading firms (Citadel, HRT, Jump, etc.)
  - **Remote / Hybrid** (26 roles) - Work from anywhere positions
- **Prestige-based ordering** within each section
- **Application status tracking** - Track Applied, Interview, Offer, Rejected
- **Resume Matcher** - AI-powered (GPT-4o) resume-to-job matching

7. Healthcare (Genentech, AstraZeneca, etc.)
8. Defense (Lockheed, BAE Systems, Shield AI, etc.)
9. Other

## Data Sources
- SimplifyJobs, speedyapply, vanshb03 (GitHub)

## Filtering Criteria
- ✅ US locations only
- ✅ Undergraduate eligible (no PhD/Masters required)
- ✅ No duplicates (based on application URL)
